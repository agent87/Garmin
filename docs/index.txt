Garmin Data Analytics's!
========================

Welcome to Garmin Data Analytics documentation. 
This is a library developed for garmin sport watches owners who utilise Linux O.S or \ 
other unsupported systems. The aim of this library is automated Data synchronisation \
across cloud or local machine plus also provide data analytics insights for the athlete \
based on data science facts.
